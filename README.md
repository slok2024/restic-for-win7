Restic 是一个现代化的、开源的备份程序。它的核心目标是：快速、高效、安全。

与传统的“压缩包备份”或简单的“复制粘贴”不同，Restic 引入了类似 Git（版本控制）的概念，将每一次备份称为一个 Snapshot（快照）。你可以随时穿越回任何一个时间点的快照，获取当时的文件状态。
